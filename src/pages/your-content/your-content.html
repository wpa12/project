Transponder

Transponder is a radio device on an aircraft that emits a signal and takes a 4-digit input provided by an air traffic controller called a squawk code

Taxiway

A taxiway is a paved area, consisting of a yellow line, an aircraft must use to navigate around an airport

Transponder Codes

7000 - conspicuity
7500 - Hijack
7600 - Communications failure code
7700 - Emergency code
7777 - Military interception code

Runway

Strip of tarmac that an aircraft uses for take-off and landing
